progress bar

## instruction

*supported promised style*<br>
*supported AMD*

```javascript
var progress = window.progressBar();

// start
progress.start()

// stop
progress.stop()

// set argument between 0 and 1
progress.set(doubleNumber)

// done
progress.done()
```
